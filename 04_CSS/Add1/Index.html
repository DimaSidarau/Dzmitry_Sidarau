Форматы файлов.

.DOC или .doc (от англ.document) - расширение имени файла, используемое для файла
предcтавляющих текст, с разметкой или без. Расширение DOC часто использовалось для
предоставления простых текстовых файлов без форматирования, однако позже стало
использоваться для двоичных форматов с разметкой.
Документ MS Word

Portable document format (PDF) - межплатформенный формат электронных документов,
разработанный фирмой Adobe Systems с использованием ряда возможностей языка
Postscript.
Документ PDF

ZIP - популярный формат сжатия данных и архивации файлов. Файл в этом формате
обычно имеет расширение zip и хранит в сжатом или несжатом виде один или несколько
файлов, которое можно из него извлечь путем распаковки с помощью специальной
программы. Архив
